Java Vector GUI

ApplicationGraphical User Interfaces
PlatformsJava, Android
ReleaseBeta, 2012
BenefitsScalable Vector GUI
ExtensibilityPlatform, Application

Description

Vector is a software framework for the development of graphical user interfaces that rebuild and deploy to multiple Java platforms.

Scalable vector GUI is a solution to problems with variations in display size, or runtime visualization geometry. For mobile devices, variations in display size commonly present a deployment problem. On the desktop, scaling visual geometry over a relatively large range may expand presentation and productivity possibilities.

The "1.0 Beta" release served a specific need that was met by that target, while the "1.0.6" branch is a work in progress to develop the set of application widgets.
