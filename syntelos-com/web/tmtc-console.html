Telemetry and Telecommand Console

ApplicationSimple Industrial Process Control
PlatformsLinux, Mac, Windows, Embedded, Android
Release(2014)
TM/TCEthernet TCP/IP, Serial RS-232
BenefitsVisualization, Recording, Customization
ExtensibilityConfiguration, Scripting, Drivers

Description

The TM/TC Console is a software application for the visualization and recording of scientific and industrial processes which are measured or controlled with TM/TC instrumentation.

The Console has been designed to support a range of process work station requirements and objectives with an installation and configuration that is simple and intuitive.
