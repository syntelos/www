Gap Data

ApplicationPerformant Data Storage & Retrieval
PlatformsGoogle App Engine - Java
ReleaseAlpha, 2011
BenefitsProductivity, Performance, Reproducibility
ExtensibilityObjects, Templates

Description

Gap Data is a software framework for the transparent generation of performant Data Objects for the GAE/Java platform.

Data Objects are declared in ODL for the generation of application data and user source code. The approach brings productivity, performance, and reproducibility to GAE/Java application development.

Integrated services for Representational State Programming of Web Services (REST) includes access control, templating, and data binding facilities.
