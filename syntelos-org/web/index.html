Syntelos is a collection of open source projects in various stages of development.

Some are server side, including proto-streamer, syntelos-sx, gap-data and pisces-graphics. Others are client side including java-vector and fv3. And some are context independent, like jela, xma, and json.

Each project represents work on a few simple ideas for a more interesting and powerful web.

Ultra light space flight is more than a software project: Fabrication. The plan is to develop some ion beams, eventually.

The Cognitive Profile Inventory is in open source at cpi.googlecode.com. A cognitive profile can inform heuristic user interfaces and other processes. The CPI includes a data API.
